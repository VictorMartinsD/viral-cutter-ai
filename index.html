Como pegar sua chave da API

  1. 1. Acesse o Google AI Studio em aistudio.google.com.
  2. 2. Entre com sua conta Google e abra a seção Get API key.
  3. 3. Clique em Criar chave de API e copie a chave gerada clicando no ícone dos papéis.
  4. 4. Volte para o Cutter.ai e cole a chave no campo acima.
Compartilhar sua chave pode permitir uso indevido da sua conta, consumo de limites de API, custos inesperados e acesso não autorizado às suas operações. Trate essa chave como uma senha: mantenha privada e atualize imediatamente se suspeitar de exposição.
